# Figma Assignment UI

A pixel-perfect React JS + Tailwind CSS implementation of the provided Figma dashboard design.  
The project focuses on clean component architecture, responsive layouts, reusable UI components, and modern frontend development practices.

---

# Live Features

- Pixel-perfect UI implementation
- Responsive dashboard layout
- Component-based architecture
- Interactive coupon selection using React state
- Smooth hover & focus animations
- Tailwind CSS custom styling
- Mobile, Tablet, and Desktop responsive support
- Professional folder structure
- Clean and scalable codebase

---

# 🛠️ Tech Stack

- React JS
- Tailwind CSS
- Vite
- React Icons
